一、热力学基本概念

温度与热平衡

温度：衡量系统能量传递趋势的物理量，通过温度计测量（如汞温度计、电阻温度计）。

热平衡：两系统通过热传递达到相同温度，满足第零定律：若 A 与 C、B 与 C 热平衡，则 A 与 B 热平衡。

系统分类

  • 孤立系统：与外界无能量、物质交换。
  • 封闭系统：仅交换能量。
  • 开放系统：交换能量和物质。

温标

  • 开尔文（K）：绝对温标，0 K 为绝对零度（不可通过有限步骤达到，第三定律）。
  • 摄氏度（℃）：公式待补充。
  • 华氏度（℉）：公式待补充。

二、热力学第一定律（能量守恒）

表达式

\(\Delta U = Q - W\)

\(\Delta U\)：系统内能变化。

\(Q\)：吸收的热量（正）或释放的热量（负）。

\(W\)：系统对外做功（正）或外界对系统做功（负）。

应用场景

  • 等容过程：\(W = 0\)，\(Q = \Delta U\)。
  • 等压过程：\(W = P\Delta V\)，\(Q = \Delta U + P\Delta V\)。
  • 绝热过程：\(Q = 0\)，\(\Delta U = -W\)。

三、热力学第二定律（熵与方向性）

克劳修斯表述

热量不能自发从低温传向高温而不引起其他变化。

开尔文 - 普朗克表述

无法制造效率为 100% 的热机（即不可能从单一热源吸热全转为功）。

熵（Entropy, \(S\)）

定义

\(dS = \frac{\delta Q_{\text{可逆}}}{T}\)

是状态函数，衡量系统无序度。

统计意义

\(S = k_B \ln W\)（\(W\)为微观态数，玻尔兹曼公式）

孤立系统熵增原理

自发过程中，\(\Delta S_{\text{孤立}} \geq 0\)（可逆过程取等号，不可逆过程熵增加）。

典型熵变计算

  • 理想气体自由膨胀：\(\Delta S = nR\ln\frac{V_f}{V_i}\)。
  • 热传导 / 混合过程：\(\Delta S = mc\ln\frac{T_{\text{终}}}{T_{\text{初}}}\)（积分求和）。

四、热机与制冷机

卡诺循环（可逆热机）

四过程

等温膨胀、绝热膨胀、等温压缩、绝热压缩。

效率

\(\eta_{\text{卡诺}} = 1 - \frac{T_L}{T_H}\)（仅取决于高低温热源温度）

制冷系数（COP）

  • 制冷模式：\(\text{COP}_{\text{制冷}} = \frac{T_L}{T_H - T_L}\)。
  • 制热模式：\(\text{COP}_{\text{制热}} = \frac{T_H}{T_H - T_L}\)。

实际热机

  • 奥托循环（汽油机）：效率取决于压缩比，\(\eta = 1 - \frac{1}{(V_1/V_2)^{\gamma - 1}}\)。
  • 狄塞尔循环（柴油机）：基于等压燃烧过程，效率低于卡诺循环。

五、气体动理论

理想气体假设

分子间作用力可忽略，碰撞为弹性碰撞。

状态方程

\(PV = nRT = Nk_BT\)（\(k_B\)为玻尔兹曼常数）

分子动能与速度

平均动能

\(\langle E_k \rangle = \frac{3}{2}k_BT\)（单原子气体）

方均根速度

\(v_{\text{rms}} = \sqrt{\frac{3RT}{M}}\)（\(M\)为摩尔质量）

麦克斯韦速度分布

\(f(v) = 4\pi\left(\frac{m}{2\pi k_BT}\right)^{3/2} v^2 e^{-mv^2/(2k_BT)}\)

描述分子速度分布概率。

自由度与比热容

  • 单原子气体：自由度\(f = 3\)，\(C_V = \frac{3}{2}R\)。
  • 双原子气体：自由度\(f = 5\)（含转动），\(C_V = \frac{5}{2}R\)。
\(C_P = C_V + R\)，比热容比\(\gamma = \frac{C_P}{C_V}\)

六、第三定律与其他

第三定律

无法通过有限步骤使系统温度降至绝对零度（\(T = 0\)）。

实际气体修正

\(\left(P + \frac{aN^2}{V^2}\right)(V - Nb) = Nk_BT\)

引入分子体积（\(b\)）和引力（\(a\)）修正。

七、关键公式总结

概念 公式
理想气体状态方程 \(PV = nRT\)
卡诺效率 \(\eta = 1 - \frac{T_L}{T_H}\)
熵变（可逆过程） \(\Delta S = \int\frac{\delta Q}{T}\)
玻尔兹曼熵公式 \(S = k_B \ln W\)
方均根速度 \(v_{\text{rms}} = \sqrt{\frac{3RT}{M}}\)
摩尔定容热容 \(C_V = \frac{f}{2}R\)（\(f\)为自由度）
