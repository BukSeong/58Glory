一、简谐振动 (Simple Harmonic Motion, SHM)

1. 基本定义与方程

运动方程：

\[ x(t) = A\cos(\omega t + \phi) \]

  • \(A\)：振幅
  • \(\omega = \sqrt{k/m}\)：角频率
  • \(\phi\)：初相位

动力学方程：

\[ \frac{d^2x}{dt^2} + \omega^2 x = 0 \]

2. 能量分析

总机械能：

\[ E = \frac{1}{2}kA^2 = \frac{1}{2}m\omega^2A^2 \]

动能与势能关系：

\[ K = \frac{1}{2}m\omega^2(A^2 - x^2), \quad U = \frac{1}{2}kx^2 \]

3. 重要现象

阻尼振动：

\[ x(t) = Ae^{-\beta t}\cos(\omega' t + \phi) \]

\[ \omega' = \sqrt{\omega_0^2 - \beta^2} \]

共振：当驱动力频率接近系统固有频率时，振幅急剧增大

二、机械波

1. 基本概念

行波方程：

\[ y(x,t) = A\cos(kx \mp \omega t) \]

  • \(k = 2\pi/\lambda\)：波数
  • \(\omega = 2\pi f\)：角频率
  • \(v = \lambda f = \omega/k\)：波速

波动微分方程：

\[ \frac{\partial^2 y}{\partial x^2} = \frac{1}{v^2}\frac{\partial^2 y}{\partial t^2} \]

2. 波的特性

叠加原理：波在相遇区域线性叠加

驻波条件

  • 固定端反射：节点形成
  • 自由端反射：波腹形成
  • 弦线驻波频率：

\[ f_n = \frac{nv}{2L} \quad (n=1,2,3,\dots) \]

3. 多普勒效应

观测频率公式：

\[ f' = f \frac{v \pm v_o}{v \mp v_s} \]

  • \(v\)：波速
  • \(v_o\)：观察者速度
  • \(v_s\)：波源速度
  • 相向运动时分子取+，分母取-

三、关键对比与记忆点

概念 振动（SHM） 波动（行波）
能量形式 动能与势能周期性转换 能量随波传播
相位传播 质点相位固定 相位随空间位置变化
数学描述 时间函数 \(x(t)\) 时空函数 \(y(x,t)\)
典型系统 弹簧振子、单摆 弦线、声波、光波
方程类型 常微分方程（ODE） 偏微分方程（PDE）

重要提醒

1. 简谐振动中，当位移最大时，动能为零，势能最大

2. 行波方程中的负号表示波向右传播，正号表示向左传播

3. 多普勒效应公式中，分子和分母的符号取决于观察者和波源的相对运动方向

四、典型例题

例题1：弹簧振子相位计算

题目：振子从平衡位置向正方向运动时开始计时，求 \(t = T/8\) 时的相位角。

解答：

\[ \phi = \omega t = \frac{2\pi}{T} \cdot \frac{T}{8} = \frac{\pi}{4} \]

例题2：弦线驻波频率

题目：长 \(L=2\ \text{m}\) 的弦线，波速 \(v=40\ \text{m/s}\) ，求前三阶驻波频率。

解答：

\[ f_1 = \frac{40}{2 \times 2} = 10\ \text{Hz} \]

\[ f_2 = \frac{2 \times 40}{2 \times 2} = 20\ \text{Hz} \]

\[ f_3 = \frac{3 \times 40}{2 \times 2} = 30\ \text{Hz} \]

例题3：复合弹簧系统

题目：质量 \(m\) 的物体连接两个并联弹簧（劲度系数 \(k_1\) 和 \(k_2\) ），求系统振动周期。

解答：

等效劲度系数： \( k_{\text{eq}} = k_1 + k_2 \)

振动周期： \( T = 2\pi\sqrt{\frac{m}{k_1 + k_2}} \)

例题4：多普勒效应

题目：警车以 \(30\ \text{m/s}\) 驶向静止观察者，鸣笛频率 \(800\ \text{Hz}\) ，声速 \(340\ \text{m/s}\) 。求观察者接收频率。

解答：

\[ f' = \frac{v}{v - v_s}f_0 = \frac{340}{340-30} \times 800 \approx 877.4\ \text{Hz} \]
