一、基本原理

1.1 爱因斯坦公设

  1. 相对性原理：物理定律在所有惯性系中形式相同
  2. 光速不变原理： \(c = 3 \times 10^8 \text{m/s}\) （真空中）

1.2 关键结论

  • 同时性的相对性
  • 速度上限 \(v \leq c\)

二、时空变换

2.1 洛伦兹变换

\[ \begin{cases} x' = \gamma (x - vt) \\ t' = \gamma \left( t - \dfrac{vx}{c^2} \right) \end{cases} \quad \gamma = \dfrac{1}{\sqrt{1 - v^2/c^2}} \]

2.2 速度叠加

\[ w = \frac{u + v}{1 + \dfrac{uv}{c^2}} \]

2.3 长度与角度

  • 长度收缩： \(L = L_0 \sqrt{1 - \beta^2 \cos^2\theta_0}\)
  • 角度变换： \(\tan \theta = \gamma \tan \theta_0\)

三、相对论效应

3.1 时间膨胀

\[ \Delta t = \gamma \Delta \tau \]

3.2 双生子佯谬

加速参考系时间流逝更慢

3.3 μ子衰变

观测系 现象 公式
地面系 时间膨胀 \(t_{1/2}^{\text{(earth)}} = \gamma t_{1/2}\)
μ子系 长度收缩 \(L' = L_0 / \gamma\)

四、相对论动力学

4.1 动量与能量

物理量 公式
动量 \(\vec{p} = \gamma m_0 \vec{v}\)
总能量 \(E = \gamma m_0 c^2\)
动能 \(K = (\gamma - 1)m_0 c^2\)

4.2 质能等价

\[ \Delta m = \frac{\Delta E}{c^2} \]

4.3 四维动量

\[ \mathbf{P} \cdot \mathbf{P} = \left( \frac{E}{c} \right)^2 - p^2 = (m_0 c)^2 \]
