力学

运动学

匀加速运动
\[v = v_0 + at, \quad x = x_0 + v_0 t + \frac{1}{2} a t^2\]

适用条件：恒定加速度

圆周运动
\[a_c = \frac{v^2}{r} = \omega^2 r \quad (\text{向心加速度})\] \[\omega = \frac{2\pi}{T} \quad (\text{角速度与周期关系})\]

动力学

牛顿第二定律
\[\sum \mathbf{F} = m\mathbf{a}\]

分量形式： \(F_x = ma_x\)

摩擦力
\[f_{\text{静}} \leq \mu_s N, \quad f_{\text{动}} = \mu_k N\]

能量与动量

动能定理
\[W_{\text{合}} = \Delta K = \frac{1}{2}mv^2 - \frac{1}{2}mv_0^2\]
动量守恒
\[\sum p_i = \sum p_f \quad \text{（系统不受外力时）}\]

刚体力学

转动惯量
\[I = \int r^2 dm\]

常用：实心球 \( I = \frac{2}{5}mr^2 \)

角动量守恒
\[\sum \tau_{\text{外}} = 0 \Rightarrow L = I\omega = \text{常数}\]

振动与波动

简谐振动

运动方程
\[\ddot{x} + \omega^2 x = 0 \quad \Rightarrow \quad x(t) = A\cos(\omega t + \phi)\]

周期： \(T = 2\pi/\omega\)

弹簧振子
\[\omega = \sqrt{\frac{k}{m}}, \quad E = \frac{1}{2}kA^2\]

波动方程

行波方程
\[y(x,t) = A\cos(kx \mp \omega t) \quad \text{（ - 表示向右传播）}\]
驻波条件
\[y(x,t) = 2A\cos(kx)\cos(\omega t) \quad \text{（节点间距 } \Delta x = \lambda/2\text{）}\]
多普勒效应
\[f' = \frac{v \pm v_o}{v \mp v_s}f \quad \text{（观察者接近源时取 + ）}\]

相对论

狭义相对论

时间膨胀
\[\Delta t = \frac{\Delta t_0}{\sqrt{1 - \frac{v^2}{c^2}}}\]

其中 \(\Delta t_0\) 是固有时间，\(v\) 是相对速度，\(c\) 是真空中的光速。

长度收缩
\[L = L_0\sqrt{1 - \frac{v^2}{c^2}}\]

其中 \(L_0\) 是固有长度。

质能关系
\[E = mc^2\]

其中 \(m\) 是物体的质量，\(E\) 是物体的能量。

相对论动量
\[p = \frac{mv}{\sqrt{1 - \frac{v^2}{c^2}}}\]

热力学

热力学第零定律

热平衡

如果两个热力学系统中的每一个都与第三个热力学系统处于热平衡(温度相同)，则它们彼此也必定处于热平衡。

热力学第一定律

能量守恒
\[\Delta U = Q - W\]

其中 \(\Delta U\) 是系统内能的变化，\(Q\) 是系统吸收的热量，\(W\) 是系统对外界做的功。

热力学第二定律

克劳修斯表述

热量不能自发地从低温物体转移到高温物体。

开尔文 - 普朗克表述

不可能从单一热源吸取热量，并将这热量完全变为功，而不产生其他影响。

熵增原理
\[\Delta S \geq 0\]

对于孤立系统，熵永不减少。

注：公式中符号遵循国际单位制（SI），如 \(v\) 为速度（m/s）， \(m\) 为质量（kg）， \(k\) 为劲度系数（N/m）。
