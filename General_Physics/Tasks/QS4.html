Question Solve 4 解题策略总结

1. 第一定律为核心方程

对任意过程：ΔU = Q - W（系统吸热 Q>0，对外做功 W>0）
理想气体内能仅取决于温度：ΔU = nCᵥΔT

2. 过程特性定解法

等容：W=0 → Q = ΔU = nCᵥΔT
等压：W = PΔV → Q = nCₚΔT
等温：ΔU=0 → Q = W = nRT ln(V₂/V₁)
绝热：Q=0 → ΔU = -W，结合 PV^γ = const（γ = Cₚ/Cᵥ）

3. 循环过程抓净功与效率

净功 W_net = 循环包围的面积（PV 图）
效率 η = W_net / Q_in（Q_in 为总吸热量）

4. 熵变计算选可逆路径

实际过程不可逆时，设计可逆过程连接初末态
ΔS = ∫dQ_rev / T（如等温过程 ΔS = Q_rev / T）
